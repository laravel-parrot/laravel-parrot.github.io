# installation

```bash
# 1.install
composer require laravel-parrot/laravel-parrot-setup:dev-master
# 2.create parrot project
php artisan laravel-parrot:parrot
# 3.parrot setup with laravel
php artisan laravel-parrot:setup
# 4.composer update
composer update -vv
# 5.composer autoload
composer dump-autoload
# 6.setup parrot provider
php artisan laravel-parrot:provider
# done...<parrot>
```
